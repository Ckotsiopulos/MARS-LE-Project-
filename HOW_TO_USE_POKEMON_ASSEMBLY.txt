===========================================
HOW TO USE POKEMON ASSEMBLY IN MARS
===========================================

IMPORTANT: You MUST enable the language first!

STEP 1: Open MARS
   - Launch mars.jar

STEP 2: Enable Pokemon Assembly Language
   - Go to: Tools → Language Switcher
   - In the dropdown menu, click "Pokemon Assembly"
   - It should highlight in blue when selected
   - Close the Language Switcher window

STEP 3: Open Your Test File
   - File → Open
   - Select your .asm file (e.g., quick_test.asm)

STEP 4: Assemble and Run
   - Click "Assemble" button (or press F3)
   - If successful, click "Run" (or press F5)
   - Check the "Run I/O" tab for battle preview output
   - Check the "Registers" tab to see HP, status, etc.

===========================================
TROUBLESHOOTING
===========================================

If you get "not a recognized operator" errors:
  ✓ Make sure you selected "Pokemon Assembly" in Language Switcher
  ✓ Restart MARS completely (close and reopen)
  ✓ Make sure PokemonAssembly.jar is in:
    mars/mips/instructions/customlangs/PokemonAssembly.jar

If "Pokemon Assembly" doesn't appear in Language Switcher:
  ✓ Restart MARS
  ✓ Check that the JAR file exists in the correct location
  ✓ Check MARS console for error messages
